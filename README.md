# 2) The Core Tool: "What Did You Eat?"

Users input:

-   meals eaten (daily or weekly)
-   grocery items purchased
-   takeout meals
-   beverages/snacks
-   location (city / postal code)
-   household size
-   dietary style
-   cooking frequency

Example:

Monday: - Starbucks latte - breakfast sandwich - chicken salad wrap -
pasta + jar sauce + chicken

Tuesday: - leftovers - Chipotle bowl - snacks

The tool analyzes patterns.

## A) Cost Snapshot

-   Estimated weekly food spend

------------------------------------------------------------------------

## B) Creative Cost Reduction Suggestions

Suggestions are behavior-aware and realistic.

### 1) Behavior Tweaks

-   Reduce coffee purchases → brew at home 3x/week
-   Reduce single-serve lunch purchases
-   Combine leftover usage

### 2) Ingredient Swaps

-   Chicken breast → thighs
-   Fresh out-of-season produce → frozen
-   Shredded cheese → block cheese
-   Name-brand → store-brand

### 3) Structure Tweaks

-   Double dinner to eliminate 2 takeout lunches
-   Batch-cook once → eat 3 times
-   Ingredient stacking across meals

### 4) Frequency Adjustments

-   Eat out 4x → reduce to 2x
-   Keep lifestyle intact while lowering cost

------------------------------------------------------------------------

# 4) Optimization Modes

## 1) Light Adjustment Mode

## 2) Moderate Optimization Mode

## 3) Aggressive Savings Mode
